# Bookaholic    

This project is a simple E-commerce, created using React and Firebase.


# Built with
`React 17.0.2`
`React-router-DOM 6.2.2`
`React-boostrap 5.1.3`
`Firebase 9.6.10`
`Lottie-react 2.2.1`
`Rc-rate 2.9.1`
`React-toastify 8.2.0`
`Sweetalert 2.1.2`
`Netlify`


# Installation

1. Clone the repo
https://github.com/HiZaniolo/Bookaholic.git

2. Install NPM packages
`npm install`

3. Run the prject
`npm start`
